# 📦 Inventory Management System (PHP Project)

This repository contains a **Stock Management System** built using **PHP**. It's a simple web-based inventory project designed to help users track products, manage stock levels, and organize inventory data efficiently.

---

## 🧾 About

This project is a basic **Inventory / Stock Management System** implemented in PHP. It allows users to:

- Add new products
- Update product information
- Track available stock
- Remove products when necessary

It’s ideal as a beginner-friendly PHP project, college assignment, or foundational system for learning database-driven web development.

---

## 🛠️ Features

✔️ Add, edit, and delete product entries  
✔️ Track stock quantities  
✔️ Simple and clean user interface  
✔️ Easy to customize and extend

## 🚀 Getting Started

To run this project locally, follow the steps below:

### 📌 Prerequisites

Make sure you have:

- PHP installed (version 7.x or higher recommended)
- A web server like **Apache** (e.g., via XAMPP, WAMP, MAMP)
- MySQL or MariaDB database

---

### 🧩 Installation

1. **Clone the repository**


git clone https://github.com/22103151-ship-it/php_project.git


2. **Move the project to your web server folder**
e.g., htdocs for XAMPP or www for WAMP.

3. **Create a database**

Use phpMyAdmin or the MySQL console:
CREATE DATABASE inventory_system;

4. **Update the database config**
Open the database config file (e.g., config.php) and set your DB credentials:

DB_HOST = localhost
DB_NAME = inventory_system
DB_USER = root
DB_PASS = (your password)

5. **Run the application**
Open your browser and go to:

http://localhost/php_project
